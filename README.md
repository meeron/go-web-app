Go web-app
======

Sample Go web application

Prerequisite
-----
* [Go >= v1.19](https://go.dev/dl/)
* [PostgrSQL](https://www.postgresql.org/download/)

Install
-----
* Install Go and PostgreSQL database engine
* Clone repository
```
git clone git@github.com:meeron/go-web-app.git
```
* Install dependencies
````
go install
````

Usage
-----
````
go run .
````
